# Useful AI Tools
### Meta Tools
- [Artificial Analysis](https://artificialanalysis.ai/): Performance and cost analysis of models and API providers
- [OpenRouter](https://openrouter.ai/): Unified LLM API provider; helps route requests to best (cheapest, fastest, highest limits, etc.) provider 

### Agents
*Need to explore more; early agentic tools were too error prone to be easily useful*

### Chat
- [ChatGPT](https://chat.openai.com/): The OG; now has free access to top model and GPT store (but can't create your own GPTs without subscription)
- [Claude](https://claude.ai/new): Free access to Claude Sonnet 3.5. Fewer customization options than ChatGPT but provides "Artifacts" (separate windows/files/etc. generated within chats) and "Projects" (paid feature)
- [Le Chat](https://chat.mistral.ai/chat): Chat access to Mistral models with some additional features that are NOT paywalled (agents/GPTs, web search, etc.)
- [Pal Chat (iOS)](https://apps.apple.com/us/app/pal-chat-ai-chat-client/id6447545085): iPhone/iPad chat app that can be connected to many LLM providers via API keys
- [LibreChat](https://github.com/danny-avila/LibreChat): Open source ChatGPT clone with ability to connect to multiple API providers

### Prompting
- [ChatGPT Prompting](https://mitenmit.github.io/gpt): Basic tool to guide you through elevating chat prompts
- [System Prompts](System%20Prompts): System prompt framework I developed, including template and custom system profiles.

### Development
- [Aider](https://github.com/paul-gauthier/aider): AI pair programming in the terminal; can edit and commit (no copy/paste) files on its own.
- [Replit](https://replit.com/): TBD
### RAG
- [ChromaDB](https://www.trychroma.com/): Open source embedding database

### Analysis
- [Julius.ai](https://julius.ai): Data analysis chat bot
- [Mapify](https://mapify.so/app): Mind maps from anything (prompt, docs, webpage, YouTube, etc.)
- [Fabric](https://github.com/danielmiessler/fabric): CLI-friendly analysis tool using carefully crafted, markdown, system prompts and helper tools (like a YouTube link to transcription scraper) to make common LLM use cases repeatable  

### Research
- [Consensus](https://consensus.app/): LLM powered question answering based on research papers. Also has a GPT in ChatGPT Plus
- [Perplexity](https://www.perplexity.ai/):  LLM powered question answering with built in citation based on web, video, and image search
- [NotebookLM](https://notebooklm.google.com): *WIP*
